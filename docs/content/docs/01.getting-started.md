## Installation

::prose-pm-install{name="nuxt-unlayer"}
::

## Setup

Register the module in your `nuxt.config.ts`:

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  modules: ["nuxt-unlayer"],
});
```

That's it. The module automatically:

- Injects the Unlayer embed script into your app's `<head>`
- Registers the `<EmailEditor>` component globally as a **client-only** component

::prose-callout{variant="info" title="Client-only rendering"}
`EmailEditor` is registered with `mode: 'client'` and will never run on the server. You do not need to wrap it in `<ClientOnly>` yourself.
::

## Basic usage

```vue [app.vue]
<template>
  <div style="height: 100vh">
    <EmailEditor @ready="onReady" />
  </div>
</template>

<script setup lang="ts">
  import type { EditorInstance } from "#unlayer/props";

  const editor = shallowRef<EditorInstance>();

  const onReady = (instance: EditorInstance | undefined) => {
    editor.value = instance;
  };
</script>
```

::prose-callout{variant="warning" title="Height is required"}
The editor fills 100% of its parent's height. Give the parent a fixed or calculated height — `100vh`, `calc(100dvh - 64px)`, etc. — or the editor will render with zero height.
::

## TypeScript

All types are re-exported from `#unlayer/props` so you only need one import:

```ts
import type {
  EditorInstance,  // the live editor reference returned by @ready
  EditorProps,     // full config accepted as component props (= UnlayerOptions)
  JSONTemplate,    // saved design JSON
  ExportHtmlResult,
  MergeTags,
  Features,
  // ... and more
} from "#unlayer/props";
```
